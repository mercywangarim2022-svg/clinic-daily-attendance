# Clinic Daily Attendance Tracker

This Excel project helps a clinic record and track daily patient visits. It captures the date, age, and service for each patient, and shows daily totals. It makes clinic work easier by keeping records organized, clear, and easy to update for reporting and planning.

## Features

- Record daily patient visits  
- Track patient age and service type  
- Calculate daily totals automatically  
- Easy to use, beginner-friendly  
- Helps with planning and reporting

## How to Use

1. Open the `ClinicAttendance.xlsx` file in Excel.  
2. Enter the date for each day in the designated column.  
3. Add patient information: age and service received.  
4. Use the total counts and summary tables to track trends.  

## Benefits

- Keeps patient records organized  
- Reduces errors from manual tracking  
- Provides quick overview of daily clinic activity  
- Supports decision-making for clinic staff
